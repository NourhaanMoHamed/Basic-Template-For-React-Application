🚀 Modern React Stack Starter Template
A high-performance, developer-first React 19 + TypeScript boilerplate designed for building scalable, production-ready web applications. This template follows a Feature-based Architecture to ensure clean code and easy maintainability.

🛠 Tech Stack & Tools
Framework: Vite (Lightning fast builds)

Language: TypeScript (Strictly typed for reliability)

Styling: Tailwind CSS v3 with OKLCH Color Palette support.

UI Components: shadcn/ui (Fully accessible and customizable).

Form Management: React Hook Form & Zod (Schema-driven validation).

Icons: Lucide React.

Animation: Framer Motion.

📂 Folder Structure (Clean Architecture)
This project uses a modular folder structure to keep logic and UI separated:

src/features/: Encapsulates feature-specific logic (Auth, Dashboard, etc.).

src/components/ui/: Reusable, atomic UI components (shadcn).

src/services/: Centralized API calls and external integrations.

src/store/: State management (Context API / Zustand / Redux).

src/lib/: Custom library configurations (Axios, utils).
